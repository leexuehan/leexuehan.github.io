## 剑指offer面试题系列


> 题目：请完成一个函数，输入一个二叉树，该函数输出它的镜像。

### 分析

所谓镜像，就是比如下面这样：

		    1         1
		
		  2   3     3    2

相当于根节点保持不动，将左右子树进行**递归调换**。

### 代码实现

    public static void MirrorTree(treeNode root) {
		if(root == null)
			return;
		MirrorTree(root.left);
		MirrorTree(root.right);
		
		treeNode tmp;
		tmp = root.left;
		root.left = root.right;
		root.right = tmp;
	}

### 总结

算是剑指offer里面做的最轻松的一题了，但前提是要想到**递归**这种必杀器。
